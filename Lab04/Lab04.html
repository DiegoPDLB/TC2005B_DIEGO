Preguntas laborario 04

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Semejanzas:

  • Ambos tienen una sintaxis similar.
  • Son populares en la programación.
  • Soportan la programación orientada a objetos.
Diferencias
  • Java es compilado, mientras que JavaScript es interpretado.
  • Java se usa principalmente en aplicaciones de escritorio y empresariales, mientras que JavaScript es fundamental en el desarrollo web.
  • Java es fuertemente tipado, JavaScript es débilmente tipado.

¿Qué métodos tiene el objeto Date?

  • getDate(): Obtiene el día del mes.
  • getMonth(): Obitiene el mes del año.
  • getFullYear(): Obtiene el año.
  • getDay(): Obtiene el día de la semana.
  • getTime(): Obtiene el tiempo en milisegundos.

¿Qué métodos tienen los arreglos?

  • push(): Añade elementos al final del arreglo.
  • pop(): Elimina el último elemento del arreglo.
  • splice(): Permite agregar o eliminar elementos en cualquier posicion del arreglo.
  • indexOf(): Busca la primera ocurrencia de un elemento en el arreglo.
  • forEach(): Ejecutar una función para que cada elemento del arreglo.

¿Cómo se declara una variable con alcance local dentro de una función?

  • Para declarar una variable con alcance local dentro de una función en JavaScript, simplemente utiliza "var", "let", o "const" dentro del cuerpo de la función.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

  • El uso de variables globales dentro de funciones puede llevar a la contaminación del espacio global y dificultar la depuración del código.
